## 서비스 사용법
1. 사이트에서 간단하게 약속의 제목만 적으면 '초대 링크'와 '결과 링크'가 생성됩니다.
2. 약속에 참여하는 다른 분들에게 '초대 링크'를 공유하여 참여하지 못하는 시간을 입력하도록 합니다.
3. 모두가 참여한 후 '결과 링크'에 들어가 모두가 참여 가능한 시간을 확인합니다. (결과 링크에서 참여자 수도 확인하실 수 있습니다.)
