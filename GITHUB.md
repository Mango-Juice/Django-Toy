## goormIDE --- GitHub
```
<!-- Commit & Push -->
git add --all
git commit -m "커밋 내용"
git push -u origin "브랜치 명"
```     
```
<!-- Create Branch & Check Out -->
git branch "브랜치 명"
git checkout "브랜치 명"
```     
```
<!-- Pull from GitHub -->
git pull origin "브랜치 명"
```     
```
<!-- Remove Local Branch -->
git branch -d "브랜치 명"
```
